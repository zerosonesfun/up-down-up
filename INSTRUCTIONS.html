Up Down Up

This is a minimal up and down voting WordPress plugin. There are no settings. You activate the plugin and you are all set. Once activated up and down arrow voting buttons will appear on your single post pages. Only logged in users can vote.

Installation

To install, go to your WordPress admin dashboard. Go to Plugins --> Add New. Choose upload plugin at the top of the page. Upload the cream-upvote.zip file. Activate it. Done.

Customization

If you do not see the voting buttons after activating the plugin there may be a conflict with your theme's CSS (styling). Or, even if there are no issues, you may want to add custom styling. This requires some CSS knowlege.

Within the Appearance --> Customize area of your WordPress admin you may add custom CSS rules to change the look or position of the voting buttons.

The main CSS classes you could target/adjust are:

.cream-fab-container { position: fixed; bottom: 50px; right: 50px; z-index: 999; cursor: pointer; background: transparent; transition: 0.3s; padding: 5px; min-width: 4em } .cream-fab-icon, .cream-fab-counter { text-align: center; } .cream-fab-icon { font-size: 1.5rem; } .cream-icon-selected { background: linear-gradient(to right, #ff8a00 0%, #da1b60 100%); -webkit-background-clip: text; -webkit-text-fill-color: transparent; } .cream-fab-counter p { margin: 0; padding: 0; }
